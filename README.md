1- Guessing Game (Python)

Welcome to the **Guessing Game!**  
In this fun and simple Python project, the computer randomly selects a word from a predefined list based on your chosen difficulty level.  
Your task is to **guess the word correctly** with the help of hints after every incorrect guess!

---

2- Features

- Choose between Easy, Medium, or Difficul* levels  
- The computer selects a random word from the chosen level  
- Get hints showing which letters are correct and in the right place  
- Count the number of attempts you take to guess the correct word  
- Simple and interactive command-line interface

---

3- How the Game Works

1. You select a difficulty level (`easy`, `medium`, or `difficult`).
2. The program randomly chooses a word from the list of that level.
3. You start guessing the word.
4. For each wrong guess, you get a hint:
   - Correct letters stay visible.
   - Incorrect ones are shown as `_` (underscores).
5. The game continues until you guess the full word correctly.
6. Finally, it shows how many attempts you took.
